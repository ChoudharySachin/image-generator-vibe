# Tutero AI Images

## Purpose
Images featuring AI assistants or AI-powered educational tools helping students learn.

## Specifications
- **Aspect Ratio**: 1500:1060 (Landscape)
- **Recommended Resolution**: 1500 x 1060 pixels
- **Format**: PNG or JPG

## Instructions
1. Place your reference images in the `images/` folder
2. Update `description.txt` with details about the visual style, color palette, and design elements

- AI assistant characters or interfaces
- Student-AI interaction scenarios

- Consistent branding and style
