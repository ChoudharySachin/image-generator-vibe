# Classroom Activity Images

## Purpose
Images depicting students engaged in classroom learning activities and group work.

## Specifications
- **Aspect Ratio**: 1500:1060 (Landscape)
- **Recommended Resolution**: 1500 x 1060 pixels
- **Format**: PNG or JPG

## Instructions
1. Place your reference images in the `images/` folder
2. Update `description.txt` with details about the visual style, color palette, and design elements

## What to Include in Reference Images
- Students collaborating or learning
- Classroom or educational settings
- Diverse, inclusive representation
- Active learning scenarios
- Consistent visual style and mood
